# nettleship-site

A static family history website. No build step — open any `.html` file directly in a browser.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Home page — links and previews for all pages |
| `nettleship-mems.html` | Personal memoirs written in 2014 by Pat Nettleship, covering life in Rotherham from the war years through to retirement |
| `myheritage/ethnicity.html` | DNA ethnicity pie charts for David and Kathryn (MyHeritage data, December 2025) |

## Data

| File | Description |
|------|-------------|
| `myheritage/ethnicity.json` | Source ethnicity percentages for both individuals |
